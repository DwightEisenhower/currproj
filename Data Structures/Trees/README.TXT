Trees
1. One Node called root, contains any number of children
2. Each child node points to any number of its own children (x >= 0)
3. Each child must be closer to the bottom of the tree than its parent

Binary Tree
1. At most 2 children/Node

Perfect Binary Tree aka Full Binary Tree
1. Must have 2 children
2. Fills from left to right w/ no empty spaces unless unavoidable

Binary Search Tree (BST)
1. Left node must be less than the parent
2. Right node must be bigger than the parent

Terms:
1. Root - only one exists
2. Leaf - node with no children
3. A node with parents and children is an interior node
4. Every node is a root of its own sub tree
